# What is this?
2022年度新入生向けの体験入部講座で作成するwebページの配布用プログラム

# 体験入部におけるデータ配布方法
- "CodeSandbox"で検索させる
- CodeSandboxのページに入ってもらう
- Googleアカウントで SignInしてもらう
- "CreateSandbox"を押下
- "Import Project"を押下
- 入力欄に[このgitHubのURL](https://github.com/selelab/js_freshman_distribution_2022)を貼り付けてもらう
- "Import and Fork"を押下
